Projects

Dijsktra's Algorithm for Wikipedia Pages

Data Structures and Algorithms Course project in the langauge Java to create a weighted, directed graph of over 1000 websites with a similarity metric as the cost. Djikstra's algorithm is applied to find the shortest path between websites.

Turing Machine Simulator

Programming Languages Course project to implement a one-way, infinite to the right Turing Machine in the language C. The program runs for any input file which represents a Turing Machine in this format, with any number of states.

KMeans Clustering of Websites

Data Structures and Algorithms Course project in the langauge Java to sort and persistently store over 200 Wikipedia pages into clusters of similar websites using KMeans clustering.

Caesar Cipher Messaging

System's Programming project in the langauge C to create a program that can encode and decode messages, convert numbers to unary form, and reverse messages.

Social Distancing Simulator

Programming Languages Course project in the language Answer Set Programming that creates a "social distancing" environment on a two dimensional grid. The program finds a path through the grid while constantly maintaining a 6 square distance from preselected points on the grid.
